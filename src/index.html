Welcome

node-organic is proof-of-concept project about organic software development.

It's core implementation in nodejs provides sustainable evolution-like software development from July 2012.

The concept has been also successfully implemented in php and java.

node-organic.com is site for the nodejs based organic implementation. From here you can seek answers, learn and contribute.

Concept

`node-organic` is proof-of-concept about organic software development aka organic computing.

Software developed using node-organic-like concepts is able to support one or more of the following properties:

  • Self-configuration
  • Self-optimization
  • Self-protection
  • Self-reflection
